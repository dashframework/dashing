Grid By "Row"

This is an auto grid. It takes however many items are within it and spaces them to fill the area available. This method requires setting a new grid for every visual row seen below. Benefits include: Less classes, more forgiveness of incorrect layouts or wrapping divs. The awesome thing about this method is it will automatically evenly space the amount of grid items within!

Grid Halves

This is a grid that is a simple parent followed by children. Nothing else. Super clean code. Unlike the above, it can't auto space all the children you throw at it. This grid is built to operate "halves", or more specifically it starts with a base amount of 8 units. Which allows any combination of whole unit arrangements in 8 pieces. This would be awesome for a very simple page layout! It does require an additional class be applied to every element telling it how far to span. Not applying a span class will result in a "eighth" element.
