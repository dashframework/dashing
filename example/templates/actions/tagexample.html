Custom Tags

You can make custom tags. 

//Example: Custom Tags
.tag {//Basic
	&.plan-11 {
		@include tag($background-color: #FAA443, $color: $black, $border-color: #FAA443, $hover-color: $black, $hover-border-color: #FAA443)
	}
}

.tag--counter {
	#{$tag-hover-rules} {
		&.plan-11 {
			border-color: red;
		}
	}
}
