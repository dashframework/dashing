Default Radio Buttons

Use radio buttons when displaying two or more answers in which the user must select exactly one choice.

<fieldset class="column column--full">

    <input type="radio" name="dashing-radio" id="dashing-radio1" checked/>
  </div>
  <div>
    <label for="dashing-radio2" class="inline">Option 2</label>
    <input type="radio" name="dashing-radio" id="dashing-radio2"/>
  </div>
